Filter unsolicited network traffic on your home Raspberry Pi Introduction ASN An Autonomous System Number (ASN) is a unique identifier assigned to an autonomous system (AS) on the internet. An autonomous system refers to a collection of IP networks and routers under the control of a single organization that presents a common routing policy to the internet. ASNs are used in the Border Gateway Protocol (BGP) to help routers identify and communicate with different networks, ensuring the efficient routing of internet traffic. There are two types of ASNs: public and private. Public ASNs are used to route data between different networks on the internet, while private ASNs are used for internal network management.
